Lua 的各个组件全部由 C 实现，完整 Lua 源码可以裁剪后移植，调用以下宏函数即可

```
/* open default libraries */
#define luaL_openlibs(L)	luaL_openselectedlibs(L, LUA_DEFAULTLIBK, 0)
```

在自定义的 'LUA_DEFAULTLIBK' 中选择需要的库，这里我定义成了
```c
#define LUA_STDEMBEDLIBK \
	(LUA_GLIBK | LUA_MATHLIBK | LUA_STRLIBK | LUA_TABLIBK)
```

只用这四个最基础的功能，先满足最简单的需求。

需要打开
```
#define LUA_32BITS
```
来启用 Lua 的 32 位整数和 32 位浮点数支持。

与 slave 相同，也可以采用 littlefs 作为 ESP32 上的文件系统，来存储 Lua 脚本和数据文件。

找到了一个仓库 https://github.com/openLuat/LuatOS，它是一个基于 Lua 的操作系统，可以参考其对硬件资源的抽象。
