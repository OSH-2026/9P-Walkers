 UART mesh 连接

                

                    UART mesh 链路为编译期配置：
                    
UART: 1

                    
TX: GPIO37

                    
RX: GPIO38

                    
Baudrate: 1000000

                    
Format: 8N1, no flow control

                    如需调整请修改 PWOS_MASTER_MESH_UART_* 后重新烧录。
                

            


            

                
 路由器局域网 Mesh 链路 (TCP)

                

                    主机经 LAN 口接入路由器局域网，并在下方端口监听 TCP 连接；
                    另一端 TCP 透传模块或从机侧 WiFi 模块以 Client 模式连接
                    …，mesh 帧原样透传，
                    注册 / 路由 / mini9P 流程与 UART 链路完全一致。
